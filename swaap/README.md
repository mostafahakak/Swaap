# SWAAP

**Connect. Collaborate. Create Value.**

Professional business networking and AI-powered matchmaking platform. This is a **JavaScript** prototype (no backend) built with Next.js and React.

## Features

- **Landing** — Hero, how it works, features, CTA
- **Auth** — Sign up (dummy email/mobile OTP) and Log in (dummy OTP)
- **Profile** — Create, edit, view professional profile
- **Events** — Discover, filter, register (free) or pay (dummy checkout)
- **Discover** — AI-style matchmaking, filter professionals, Connect (dummy)

## Run locally

```bash
cd swaap
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech stack

- **Next.js** (App Router)
- **React**
- **JavaScript** (no TypeScript)
- **Tailwind CSS**

---

Prototype only — no backend, no real auth or payments.
